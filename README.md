# Менеджер задач

Test project "Менеджер задач" for Hexlet "php" course

## Setup
Setup SSH before clone:

git clone git@github.com:murYG/php-project-57.git
cd php-project-57

make setup

## Launch
    make start
	
## Prerequisites
    Linux, Macos, WSL
    PHP >=8.4
    Make
    Git
